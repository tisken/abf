# ABF — Anima Beyond Fantasy para Foundry VTT

Sistema no oficial de Anima Beyond Fantasy para Foundry VTT v14.

## Características

- Ficha de personaje completa con 13 pestañas
- Sistema de PDs multiclass con bloques secuenciales
- Cálculos automáticos (stats, resistencias, combate, magia, ki, psíquica)
- Tirada abierta d100 con explosiones, pifias y dobles
- Sistema de combate con diálogos de ataque/defensa
- Críticos con 4 niveles de severidad y localización
- 18 tipos de items (armas, armaduras, conjuros, poderes, ventajas...)
- Pestañas dinámicas configurables (Ki, Magia, Psíquica, Elan, Grimorios)
- Traducciones ES/EN completas
- Estilo visual tipo ficha impresa

## Instalación

En Foundry VTT → Configuración → Sistemas → Instalar sistema:

```
URL del manifiesto: [pendiente]
```

## Pestañas

### Siempre visibles
- **Resumen** — Dashboard de juego con tiradas
- **General** — Datos narrativos, inventario
- **Principal** — Características, resistencias, secundarias
- **PDs** — Puntos de Desarrollo multiclass
- **Combate** — Armas, armaduras, valores de combate
- **Personalización** — Configuración del personaje

### Configurables
- Ki, Técnicas, Místico, Psíquicos, Elan, Grimorio Magia, Grimorio Psíquica
